## 5. 遇到的问题与解决方案

### 5.1 H2 自增 ID 问题
**问题**: `Null not allowed for column "id"` - MyBatis-Plus 默认使用雪花算法，但 H2 需要自增
**解决**: schema.sql 中使用 `GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY`

### 5.2 Spring Security 拦截
**问题**: 403 Forbidden - 测试请求被安全拦截
**解决**: 创建 TestSecurityConfig 禁用 CSRF 并放行所有请求

### 5.3 ComponentScan 缺失
**问题**: 404 Not Found - 找不到 Controller Bean
**解决**: 在 LinkApplication.java 添加 `@ComponentScan` 包含 `com.qingluo.link.api`

### 5.4 embedded-redis macOS 兼容
**问题**: `Can't start redis server` - embedded-redis 在 macOS 上无法运行
**解决**: 使用 Homebrew 安装本地 Redis (`brew install redis && brew services start redis`)

### 5.5 H2 数据库状态污染
**问题**: 多个测试类共享 H2 数据库导致 ID 冲突、provider_type 冲突
**解决**:
- 为每个测试类分配唯一 ID
- provider_type 使用唯一值（如 `openai_config`）
- 使用 `@DirtiesContext` 隔离 Spring 上下文

### 5.6 sa-token 上下文问题
**问题**: `未能获取有效的上下文处理器` - 在非 Web 环境使用 StpUtil
**解决**: 使用 `@Import(TestSecurityConfig.class)` 确保 sa-token 上下文正确初始化
